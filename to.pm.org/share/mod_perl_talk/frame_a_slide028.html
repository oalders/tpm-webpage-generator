Survey of Modules for Mod_perl (e.g. Apache::.*)

  • Apache::Language - management of multilingual translation of text
  • Apache::AuthCookie - handle authentication & authorization based on cookie contents
  • Apache::AuthLDAP - authorizes against an LDAP server
  • Apache::DBI - caches database handles and shares with processes
  • Apache::MP3 - turns your Apache web server into a streaming MP3 server
  • Apache::Session - should be called "Persistent::Hash"
