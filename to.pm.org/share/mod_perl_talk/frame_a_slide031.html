Enumerated Kits (cont.)

  • Apache::ASP - http://www.apache-asp.org/
    • A Perl implementation (clone) of the ASP model for mod_perl
  • CGI::Application -
    • Event & Callback-oriented dispatch system for CGI-ish applications (doesn't depend on mod_perl)
  • HTML::Template - http://sourceforge.net/projects/html-template/
    • A templating system optimized for HTML (doesn't depend on mod_perl)
  • Apache::PageKit - http://pagekit.org/
    • A cobbled-together MVC system build upon HTML::Template & other standard Apache::* modules
