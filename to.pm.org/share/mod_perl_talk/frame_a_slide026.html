Installing mod_perl

  • There's more than one way to do it -- I'll talk about the easiest one :-)
  • Including SSL support via mod_ssl and OpenSSL
  • Including mod_python, mod_php, mod_frontpage, etc.
  • Linked statically or dynamically
  • I will demonstrate the easiest version
  • Get Apache tarball at http://www.apache.org/dist/httpd/
  • Get mod_perl tarball at http://perl.apache.org/dist/
  • Untar both, cd into mod_perl directory, and build them together like a Perl module
    • For the love of God, use a makepl_args.mod_perl file!
  • verifying that it's actually installed
    • Start up the web server and check out error_log
    • Or telnet into the web server and issue a head command
