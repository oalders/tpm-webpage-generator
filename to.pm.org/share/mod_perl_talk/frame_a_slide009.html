Benefits of Web API Programming

  • Generically, API programming provides opportunities to the programmer, but it is up to them to take the maximum advantage of them
  • Persistence -- gives opportunity for code-caching and database connections
  • Access to web server internals and internal API -- you can "do more things" than what CGI lets you
  • However, "mere mortals" don't write CGI replacement programs directly with the APIs
    • Who wants to deal with C for web applications?
    • 1,001 printf's, etc.
    • Moving away from C for CGI was what made Perl so popular in the first place
