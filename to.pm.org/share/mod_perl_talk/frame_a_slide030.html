Enumerated Kits (Only a sampling)

  • ePerl - http://www.engelschall.com/sw/eperl/
    • Perl code embedded within random text pages
    • Exists as its own executable (eperl) and as an Apache module (Apache::ePerl)
  • HTML::Embperl - http://perl.apache.org/embperl/
    • like ePerl, but includes "shortcuts" for HTML-oriented files (e.g. table row population, form variable decoding)
  • HTML::Mason - http://www.masonhq.com/
    • "Component-oriented" kit, also embeds Perl code inside of HTML-ish pages
  • OpenInteract - http://www.openinteract.org/
    • Complete application construction framework
  • AxKit - http://www.axkit.org/
    • Apache/XML kit -- copes well with XML source data, XPath, XSL Transformations, etc.
