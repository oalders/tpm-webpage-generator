Fixing A Common Misconception

  • People will often think that "mod_perl is harder than PHP", or than Cold Fusion, ASP/VB, etc.
  • Yes, but...
  • This comparing apples and oranges
  • mod_perl is a way to get at the Apache API easily
    • mod_perl is the easiest API to work with!
  • PHP, Cold Fusion, ASP/VB, etc., are "toolkits" built on top of ISAPI, NSAPI, Apache API, etc.
  • You shouldn't try to compare the two
  • Rather, there are Perl-oriented toolkits that should be compared against these
