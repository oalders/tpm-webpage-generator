What is a CGI request?

  • As people became more familiar with the web, they realized that web servers didn't have to restrict themselves to serving simple files that happened to exist on the hard drive
  • CGI is the Common Gateway Interface -- it "interfaces" the http request and the web server to a program located at the site
  • CGI defines a very simple set of programming conventions such that any program that follows them can be run through any web server that understands CGI
    • environment variables and STDIN as input
    • STDOUT as output
    • "standard URL encoding", cookies, a few other things as part of input
    • as output the program must produce an appropriate "Content-type" header and (probably) HTML
