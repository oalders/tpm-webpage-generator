Web API Programming -- The Next Approach

  • By 1995 web servers started to offer another approach beyond CGI for back-end web programming: API-oriented programming
  • Each web server has its own API
    • MSIIS / ISAPI
    • iPlanet (Netscape) / NSAPI
    • O'Reilly Website (WSAPI)
    • Apache / Apache API
  • This general class of techniques allows you to bring your program "inside" of the web server
