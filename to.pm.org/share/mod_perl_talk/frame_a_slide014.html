The Apache web server

  • Apache, son of NCSA, son of CERN
  • Leading web server by "market share" of publically available http servers since 1996 - http://www.netcraft.co.uk/survey/
  • Open Source software by Apache License (BSD variant)
  • It does what web servers are supposed to do -- respond to HTTP requests
    • Decently fast
    • Well-tested
    • Very featureful, very configurable (handles virtual hosts very well, so it is a darling of ISPs)
    • It is "correct" -- adheres to W3C HTTP protocol standards (much to the chagrin of AOL)
    • It is highly modular, which has lead to a number of 3rd-party add-ons
    • (e.g. mod_ssl, mod_php, mod_python, mod_tcl, mod_frontpage, mod_perl)
  • Apache 1.3 is a traditional forked-process model Unix daemon-style server
