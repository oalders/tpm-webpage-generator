Simple program to generate TPM index page.

  make critic # testing perl code
  make clean
  make
  make install

Prerequisites:

  Path::Class
  Readonly
  Template
  File::Find::Rule
  XML::Twig
  Date::Parse
  Perl::Critic (for make critic)
